# pdb4all
*Convert between common protein pdb formats and names*

#### Version:  0.1 - 04/05/2020

### Usage
```
  pdb4all <options> .pdb
```

### Currently supported ForceFields name conventions
  * Amber
  * Charmm
  * OPLS

### Currently supported output formats
  * Fasta
  * GROMACS
  * fDynamo
